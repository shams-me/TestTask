### City tower placer

This is a simple program that places towers in a city.

Visualisation is done with emojis. Decided to use only default libraries python instead matplotlib or seaborn.

Emojies definitions:

- 🗼 : Tower
- ❌ : Blocked grid
- 🟥 : Not covered place
- 🟩 : Covered place

### How to use

```bash 
python3 main.py
```
